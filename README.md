# clashctl2
 a cli tool for clash

## Install

### Prerequisites

- cmake
- curl
- make
- g++

```bash
# clashctl will be install to ~
make install
```



## Usage

```bash
# update clash config
~/clashctl/clashctl update <url>

# see more from help
~/clashctl/clashctl help

# once started, use the shortcuts to apply or de apply proxy for current terminal
~/clashctl/set_proxy
~/clashctl/unset_proxy
```
